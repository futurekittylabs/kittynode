Kittynode is free software, and in the spirit of that, let's build from source.

### Prerequisites

:::note
You can skip this section if you're using Nix.
:::

<Steps>

1. Install [Rust](https://www.rust-lang.org/tools/install):

    ```bash
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```

2. Install [Just](https://github.com/casey/just):

    ```bash
    cargo install just
    ```

3. Install [Bun](https://bun.sh):

    ```bash
    curl -fsSL https://bun.sh/install | bash
    ```

4. Install [Tauri system dependencies](https://tauri.app/start/prerequisites/#system-dependencies).

</Steps>

### Build from source

<Steps>

1. Clone the repo and change into it:

    ```bash
    git clone git@github.com:futurekittylabs/kittynode.git
    cd kittynode
    ```

2. Install dev tools:

    <Tabs>
      <TabItem label="Not using Nix">
        ```bash
        just install-dev-tools
        ```
      </TabItem>
      <TabItem label="Using Nix">
        ```bash
        nix develop
        ```
      </TabItem>
    </Tabs>

3. Install dependencies:

    ```bash
    bun install
    ```

4. Build all crates:

    ```bash
    just build
    ```

5. Build the desktop app:

    ```bash
    just build-app
    ```

</Steps>

### Run Kittynode

<Steps>

1. Run the CLI:

    ```bash
    just kittynode
    ```

2. Run the desktop app:

    ```bash
    just app
    ```

</Steps>
