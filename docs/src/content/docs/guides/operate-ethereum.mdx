This guide will go over how to operate the world computer with Kittynode.

## Getting hardware

To run an Ethereum node, it's best to use dedicated hardware. I would recommend the [ASUS NUC 14 Pro](https://www.newegg.com/asus-nuc-configurator/ASUS-NUC-14-Pro/56-110-266) with 64GB RAM and 4TB SSD. Minisforum and Beelink would be good options as well.

Please refer to [EIP-7870: Hardware and Bandwidth Recommendations](https://eips.ethereum.org/EIPS/eip-7870) for further guidance. Kevaundray's [post on hardware requirements](https://hackmd.io/@kevaundray/S1hUQuV4Jx) is also helpful.

## Set up an Ubuntu machine

- Install [Ubuntu desktop or server](https://ubuntu.com/download)
- Install basic utils:
  ```bash
  sudo apt update
  sudo apt install -y build-essential curl git
  ```
- Install Tailscale:
  ```bash
  curl -fsSL https://tailscale.com/install.sh | sh
  ```

### Set up SSH access over Tailscale

- Install openssh-server:
  ```bash
  sudo apt update
  sudo apt install -y openssh-server
  ```
- Enable and start the SSH service:
  ```bash
  sudo systemctl enable ssh
  sudo systemctl start ssh
  ```

### Install Docker

- Uninstall any conflicting packages:
  ```bash
  for pkg in docker.io docker-doc docker-compose docker-compose-v2 podman-docker containerd runc; do sudo apt-get remove $pkg; done
  ```

- Install Docker:
  ```bash
  sudo apt-get update
  sudo apt-get install ca-certificates curl
  sudo install -m 0755 -d /etc/apt/keyrings
  sudo curl -fsSL https://download.docker.com/linux/ubuntu/gpg -o /etc/apt/keyrings/docker.asc
  sudo chmod a+r /etc/apt/keyrings/docker.asc
  echo \
    "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.asc] https://download.docker.com/linux/ubuntu \
    $(. /etc/os-release && echo "${UBUNTU_CODENAME:-$VERSION_CODENAME}") stable" | \
    sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
  sudo apt-get update
  sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
  ```

- Verify Docker is installed:
  ```bash
  sudo docker run hello-world
  ```
